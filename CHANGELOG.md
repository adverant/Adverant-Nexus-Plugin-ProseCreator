# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-29

### Added
- Initial release of ProseCreator plugin
- Multi-format support: Novel, Screenplay, Stage Play, Comic Book, Poetry, Non-Fiction
- Living Blueprint System with self-evolving story documents
- Infinite Memory Architecture powered by GraphRAG
- 15+ humanization techniques for undetectable AI content
- WebSocket streaming for real-time generation
- 40+ REST API endpoints for comprehensive project management
- Character Bible integration with appearance tracking
- Research agent for topic integration
- Continuity and plot-hole analysis
- Multi-export formats: Markdown, PDF, EPUB, DOCX

### Technical
- Express.js HTTP server with TypeScript
- WebSocket server for real-time communication
- Neo4j integration for knowledge graphs
- Qdrant integration for vector embeddings
- PostgreSQL for primary data storage
- Redis for rate limiting and caching
